# SphericalContoursStandalone

A stand-alone version of the [Spherical Contours](https://observablehq.com/@fil/spherical-contours) observable notebook

Install:

> yarn setup

Build and serve locally:

> yarn serve

Deploy:

> yarn deploy

Update notebook or data:

> yarn notebook

> yarn data
